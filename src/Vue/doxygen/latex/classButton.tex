\doxysubsection{Detailed Description}
UI \mbox{\hyperlink{classButton}{Button}} class. 

You can select, deselect and add a callback function to this button. It benefits from all the features of the classes \mbox{\hyperlink{classUIElement}{UIElement}} and \mbox{\hyperlink{classSpriteNode}{Sprite\+Node}}.
