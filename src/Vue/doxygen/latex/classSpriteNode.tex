\doxysubsection{Detailed Description}
Base class for sprite element. 

The sprite can be animated or not. It needs to be supplied a Sprite\+::\+Properties to be displayed. Sprite\+::\+Properties contains information about the path to the Sprite texture, it\textquotesingle{}s size and whether it\textquotesingle{}s animated or not.
